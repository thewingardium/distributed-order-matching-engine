# Distributed Order Matching Engine

A distributed order matching engine built with Java Spring Boot.

## Prerequisites

- Java 17
- Maven

## Building and Running

Since this project was initialized in an environment without Maven installed, you will need to have Maven installed on your machine to build and run it.

```bash
mvn clean install
mvn spring-boot:run
```

## Structure

- `src/main/java/com/example/dome` - Source code
- `src/main/resources` - Configuration
